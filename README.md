# Piton Task

Stack:
- Next.js
- Tailwind
- TypeScript
- Redux

## 📦 Installation

### 📰 Clone the project

```bash
git clone https://github.com/emreunsal1/piton-task.git
```

### 🔻 Install Dependencies

With yarn:
```bash
cd ./piton-task && yarn
```

### 🏃‍♂️ Start the project

With yarn:
```bash
yarn dev
```
